The static directory

Basically all files that are not Django/Python code or html templates should be kept in the static directory, including css and javascript, favicons, images, and audio/video stimuli for the MoReDaT Apps.

The static directory is mirrored in organization to the MoReDaT root directory. This means that for each MoReDaT app, there is a directory in static that contains all the static for that app.

The recorded_audio directory

This is the directory where tasks that record audio data dump files.
