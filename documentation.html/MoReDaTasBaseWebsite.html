Using MoReDaT as the basis for a new website

Here are some things to consider doing if you want to adopt MoReDaT as the basis for a new website:

  • Customize urls, viwes functions, and templates in main
  • Customize the look and feel of the site with css in static/main/main.css
  • Bulk up your authentication system
    • add email field to user registration form
    • on create user set is_active to False
    • confirm email to activate

    • password reset form

Consider how you want to deploy your site into production (this is well beyond the scope of this documentation, but some things for consideration):

  • renting server space or acquiring hardware
  • which linux distro do you want to install on
  • which webserver software (Apache, Nginx, etc)
  • acquiring domain name...
  • etc...
