MOREDAT Scaffolding

In order to make MOREDAT Apps operable and testable "out of the box", it is built around the most basic parts of a Django website. This is referred to in the documentation as Scaffolding.

What is MOREDAT Scaffolding?

Scaffolding consists of:

  • two apps, main and users
