{% trans 'Instructions' %}

{% trans 'In this task, you will be shown images of miscellaneous items and you should say what you see in that image. The task consists of six 'blocks' of images, each with several images. It will be indicated at the beginning of each block which language you should name the pictures in.' %}

{% translate 'You should use a single word to describe each picture. For instance, if you are shown an image depicting a bird, you should say 'bird', *NOT 'I see a bird.' or 'There is a bird.' – one word is enough.' %}

{% translate 'After you see each image, your browser will record your voice response. Each block of images progresses on its own; you have approximately five seconds to name the image before the next image is shown. You can control the start of each block of image with a button, but all other links on the site will be disabled and you should do the entire RAN experiment in one sitting. It takes around 10 minutes if without any long pauses between trial blocks.' %}

{% translate 'Once you start the task you can not stop / pause it, go back, or start over. Please be sure you are in a suitable environment and have time to complete the task before you get started.' %}

{% translate 'Details about the experiment: ' %}{% translate ' Show' %}{% translate ' Hide' %}

{% translate 'The name of the so-called RAN experiment is an acronym for Rapid Automatized picture-Naming. You have already seen in the instructions above that it is RAPID; you are given approximately 5 seconds to produce a word corresponding to the image stimulus. It is Automatized in the sense that you, as the participant, do not have controll over the pace at which stimili are presented to you.' %}
